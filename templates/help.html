This is the NoobCash website, here you have the possibility to click on three button.
  • New transaction: It calls up the transaction.html page which allows you to make a new transaction using the sender and reciever IDs that you wanted to use. In addition, you must choose an amount to complete the transaction.
  • View last transactions: This button shows the "view.html" page that allows to display all the transactions of the last block in the blockchain.
  • Show balance: When you click on this button, you display the "balance.html" page. In this page you can see the balance of the wallet according to the id that you chose in the dropdown.
